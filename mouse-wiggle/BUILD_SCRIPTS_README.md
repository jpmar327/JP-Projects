# BUILD SCRIPTS README

This documentation explains all the build scripts and how they operate together.

## Scripts Overview

1. **installer.nsi**: Windows NSIS installer script for creating the installer.
2. **build_macos_installer.sh**: Script to create a .dmg installer for macOS.
3. **build_windows.bat**: Windows batch script for building the application and installer.
4. **build.sh**: Universal build script compatible with Windows, macOS, and Linux.

## Usage

To build the project:
- Run `build.sh` from your terminal.
- Follow platform-specific notes as indicated in the individual script files.
