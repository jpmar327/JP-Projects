# Mouse Wiggler Application

## Introduction
The Mouse Wiggler is an application designed to prevent your computer from going idle by simulating mouse movements.

## Features
- **Mouse Simulation:** Simulates mouse movement to keep your system active.
- **Custom Intervals:** Set your own intervals for mouse movement.
- **Lightweight:** Minimal impact on system performance.

## Quick Start Instructions
1. **Installation**: Download and install the application from the [release page](#).
2. **Run the App**: Launch the Mouse Wiggler.
3. **Configure Settings**: Adjust the settings according to your preferences.
4. **Start Wiggling**: Click the "Start" button to begin simulating mouse movements.

## Developer Setup
1. **Clone the Repository**: Run `git clone https://github.com/jpmar327/JP-Projects.git`
2. **Install Dependencies**: Navigate to the project directory and run `npm install`.
3. **Run the Application**: Use `npm start` to launch the application.
